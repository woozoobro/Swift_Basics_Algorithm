## 만약에 Undirected 라면? 어떻게 될까?

Edge List 같은 경우엔 뒤집은 것도(순서를 바꾼 것도) 작성해주면 됨

Adjacency Matrix 는 표의 반대방향으로 대각선을 기준으로 똑같이 작성해주면 되겠죠!

Adjacency List는 그냥 다 적어주면 됨
0 -> [2]
1 -> [3]
2 -> [0, 3, 4]
3 -> [1, 2, 5]
4 -> [2, 5]
5 -> [3, 4]
